Chainlit 支持对[消息](/concepts/message)和[步骤](/concepts/step)进行流式输出。以下是一个使用 `openai` 的示例。

<Frame caption="流式输出 OpenAI 响应">
  <img src="/images/streaming.gif" />
</Frame>

```python
from openai import AsyncOpenAI
import chainlit as cl

client = AsyncOpenAI(api_key="YOUR_OPENAI_API_KEY")


settings = {
    "model": "gpt-3.5-turbo",
    "temperature": 0.7,
    "max_tokens": 500,
    "top_p": 1,
    "frequency_penalty": 0,
    "presence_penalty": 0,
}


@cl.on_chat_start
def start_chat():
    cl.user_session.set(
        "message_history",
        [{"role": "system", "content": "你是一个乐于助人的助手。"}],
    )


@cl.on_message
async def main(message: cl.Message):
    message_history = cl.user_session.get("message_history")
    message_history.append({"role": "user", "content": message.content})

    msg = cl.Message(content="")

    stream = await client.chat.completions.create(
        messages=message_history, stream=True, **settings
    )

    async for part in stream:
        if token := part.choices[0].delta.content or "":
            await msg.stream_token(token)

    message_history.append({"role": "assistant", "content": msg.content})
    await msg.update()
```

## 集成

部分集成在更高层级也支持流式输出。

例如，要在 Langchain 中使用流式输出，只需在实例化 LLM 时传入 `streaming=True`：

```python
llm = OpenAI(temperature=0, streaming=True)
```

同时请确保向 chain 或 agent 的运行传入[回调处理器](/api-reference/integrations/langchain)。

关于最终答案的流式输出，请参阅[此处](/api-reference/integrations/langchain#final-answer-streaming)。
