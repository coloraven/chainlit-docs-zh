---
title: "Plotly"
---

`Plotly` 类允许你在聊天机器人 UI 中显示 Plotly 图表。此类接收一个 Plotly figure。

`Plotly` 元素相比 `Pyplot` 元素的优势在于它是交互式的（例如用户可以在图表上缩放）。

## 属性

<ParamField path="name" type="str">
  在 UI 中显示的图表名称。
</ParamField>

<ParamField path="display" type="ElementDisplay" optional>
  决定图表元素在 UI 中的显示方式。可选值为 "side"、"inline" 或 "page"。
</ParamField>

<ParamField path="size" type="ElementSize" optional>
  决定图表大小。仅当 `display="inline"` 时生效。可选值为 "small"、"medium"（默认）或 "large"。
</ParamField>

<ParamField path="figure" type="str">
  要显示的 `plotly.graph_objects.Figure` 实例。
</ParamField>

### 示例

```python
import plotly.graph_objects as go
import chainlit as cl


@cl.on_chat_start
async def start():
    fig = go.Figure(
        data=[go.Bar(y=[2, 1, 3])],
        layout_title_text="示例图表",
    )
    elements = [cl.Plotly(name="chart", figure=fig, display="inline")]

    await cl.Message(content="此消息包含一个图表", elements=elements).send()
```
