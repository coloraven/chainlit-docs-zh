title: "Langchain 回调处理器"
---

以下代码示例演示如何传递回调处理器：

```python
llm = OpenAI(temperature=0)
llm_math = LLMMathChain.from_llm(llm=llm)

@cl.on_message
async def main(message: cl.Message):

    res = await llm_math.acall(message.content, callbacks=[cl.LangchainCallbackHandler()])

    await cl.Message(content="你好").send()
```

## 最终答案流式输出

若在 LLM 级别启用了流式输出，LangChain 默认只会流式传输中间步骤。你可以通过向回调处理器传入 `stream_final_answer=True` 来启用最终答案的流式输出。

```python
# 可选：你还可以传入用于识别最终答案的前缀 token
answer_prefix_tokens=["FINAL", "ANSWER"]

cl.LangchainCallbackHandler(
        stream_final_answer=True,
        answer_prefix_tokens=answer_prefix_tokens,
    )
```

<Warning>
  最终答案流式输出仅适用于具有固定最终答案模式的提示词。它也不适用于
  `AgentType.STRUCTURED_CHAT_ZERO_SHOT_REACT_DESCRIPTION`
</Warning>
