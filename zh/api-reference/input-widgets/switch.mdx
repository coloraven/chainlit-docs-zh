title: "Switch"

### 属性

<ParamField path="id" type="str">
  用于从设置中检索控件值的标识符。
</ParamField>

<ParamField path="label" type="str">
  输入控件的标签。
</ParamField>

<ParamField path="initial" type="int">
  输入控件的初始值。
</ParamField>

<ParamField path="tooltip" type="str" optional>
  悬停在标签旁工具提示图标上时显示的工具提示文本。
</ParamField>

<ParamField path="description" type="str" optional>
  显示在输入控件下方的文本。
</ParamField>

### 用法

```python Code Example
import chainlit as cl
from chainlit.input_widget import Switch


@cl.on_chat_start
async def start():
    settings = await cl.ChatSettings(
        [
            Switch(id="Streaming", label="OpenAI - 流式输出 Token", initial=True),
        ]
    ).send()
    value = settings["Streaming"]

```
