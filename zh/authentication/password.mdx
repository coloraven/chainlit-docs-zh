---
title: 密码
---

`@cl.password_auth_callback` 接收登录表单中的用户名和密码。返回 `cl.User` 对象将认证用户，返回 `None` 则认证失败。

你可以对照任意服务验证凭据（自己的数据库、私有 Google 表格等）。

<Warning>
  此处应遵循常规安全最佳实践，在存储前对密码进行哈希处理。
</Warning>

## 示例

```python
from typing import Optional
import chainlit as cl

@cl.password_auth_callback
def auth_callback(username: str, password: str):
    # 从数据库获取与用户名匹配的用户
    # 并将哈希后的密码与数据库中存储的值进行比较
    if (username, password) == ("admin", "admin"):
        return cl.User(
            identifier="admin", metadata={"role": "admin", "provider": "credentials"}
        )
    else:
        return None
```
