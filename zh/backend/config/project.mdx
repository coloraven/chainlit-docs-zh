## 选项

<ParamField path="allow_origins" type="List[str]" default={["*"]}>
  允许访问应用/Copilot 的来源。
</ParamField>

<ParamField path="transports" type="List[str]" optional>
  Socket.io 客户端 transports 选项
</ParamField>

<ParamField path="user_env" type="List[str]" optional>
  每位用户使用应用时需要提供的环境变量列表。若为空，则不会向用户询问环境变量。
</ParamField>

<ParamField path="persist_user_env" type="bool" default={false}>
  是否将用户环境变量（API 密钥）持久化到数据库。启用后，用户无需在每个会话中重新输入密钥。
</ParamField>

<ParamField path="mask_user_env" type="bool" default={false}>
  是否在 UI 中使用密码类型输入框掩码显示用户环境变量。
</ParamField>

<ParamField path="lc_cache_path" type="str" optional>
  本地 langchain 缓存数据库路径
</ParamField>

<ParamField path="session_timeout" type="int" default={300}>
  连接丢失时会话保存的持续时间（秒）
</ParamField>

<ParamField path="user_session_timeout" type="int" default={1296000}>
  用户会话过期的持续时间（秒）。默认 15 天
</ParamField>

<ParamField path="cache" type="bool" default={false}>
  启用第三方缓存（例如 LangChain 缓存）
</ParamField>

## 默认配置

```toml
[project]
# 每位用户使用应用时需要提供的环境变量列表
user_env = []

# 是否将用户环境变量（API 密钥）持久化到数据库
persist_user_env = false

# 是否在 UI 中使用密码类型掩码显示用户环境变量（API 密钥）
mask_user_env = false

# 连接丢失时会话保存的持续时间（秒）
session_timeout = 3600

# 用户会话过期的持续时间（秒）
user_session_timeout = 1296000  # 15 天

# 启用第三方缓存（例如 LangChain 缓存）
cache = false

# 允许的来源
allow_origins = ["*"]
```
