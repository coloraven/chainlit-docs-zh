title: "概览"
---

运行 `chainlit run ...` 或 `chainlit init` 时会创建 `.chainlit/config.toml` 文件。它允许你配置 Chainlit 应用并启用/禁用特定功能。

你还可以在运行时通过聊天配置动态覆盖特定的 `config.toml` 变量。详情请参阅[基于配置的动态配置](/api-reference/lifecycle-hooks/on-profile-switch)。
