
## React Hooks

`@chainlit/react-client` 包提供多个 React hooks，用于无缝管理聊天应用的各个方面：

- **[`useChatSession`](#usechatsession-hook)**：管理与 WebSocket 服务器的[聊天会话](/concepts/chat-lifecycle)连接。
- **[`useChatMessages`](#usechatmessages-hook)**：管理[消息](/concepts/message)的检索与渲染。
- **[`useChatData`](#usechatdata-hook)**：访问聊天相关数据和状态。
- **[`useChatInteract`](#usechatinteract-hook)**：提供与聊天系统交互的方法。
- **[`useAuth`](#useauth-hook)**：处理[身份认证](/authentication)流程。
- **[`useApi`](#useapi-hook)**：简化 API 交互，内置数据获取和错误处理支持。

---

### `useChatSession` Hook

此 hook 负责管理与 WebSocket 服务器的[聊天会话](/concepts/chat-lifecycle)连接。

#### 方法

- **`connect`**：建立与 WebSocket 服务器的连接。
- **`disconnect`**：断开与 WebSocket 服务器的连接。
- **`setChatProfile`**：设置[聊天配置](/advanced-features/chat-profiles)状态。

#### 示例

```tsx
import { useChatSession } from '@chainlit/react-client';

const ChatComponent = () => {
  const { connect, disconnect, chatProfile, setChatProfile } = useChatSession();

  // 连接到 WebSocket 服务器
  useEffect(() => {
    connect({
      userEnv: {
        /* 用户环境变量 */
      },
      accessToken: 'Bearer YOUR ACCESS TOKEN', // 可选的 Chainlit 认证令牌
    });

    return () => {
      disconnect();
    };
  }, []);

  // 组件的其余逻辑
};
```

### `useChatMessages` Hook

`useChatMessages` hook 提供对当前聊天[消息](/concepts/message)、首次用户交互以及活动线程 ID 的访问。它利用 Recoil 进行状态管理，确保组件在状态变化时响应式更新。

#### 返回值

- **`threadId`** (`string | undefined`)：  
  当前聊天线程的标识符。
- **`messages`** (`IStep[]`)：  
  聊天[消息](/concepts/message)数组。
- **`firstInteraction`** (`string | undefined`)：  
  首次用户发起交互的内容。

#### 示例

```tsx
import { useChatMessages } from '@chainlit/react-client';

const MessagesComponent = () => {
  const { messages, firstInteraction, threadId } = useChatMessages();

  return (
    <div>
      <h2>线程 ID：{threadId}</h2>
      {firstInteraction && <p>首次交互：{firstInteraction}</p>}
      {messages.map((message) => (
        <p key={message.id}>{message.content}</p>
      ))}
    </div>
  );
};
```

### `useChatData` Hook

`useChatData` hook 提供对 React 应用中各种聊天相关状态和数据的全面访问。

#### 返回属性

- **`actions`** (`IAction[]`)
- **`askUser`** (`IAsk | undefined`)
- **`chatSettingsValue`** (`any`)
- **`connected`** (`boolean`)
- **`disabled`** (`boolean`)
- **`error`** (`boolean | undefined`)
- **`loading`** (`boolean`)
- **`tasklists`** (`ITasklistElement[]`)

#### 示例

```tsx
import { useChatData } from '@chainlit/react-client';

const ChatStatusComponent = () => {
  const { connected, loading, error, actions, askUser, chatSettingsValue } = useChatData();

  return (
    <div>
      <h2>聊天状态</h2>
      {loading && <p>正在加载聊天...</p>}
      {error && <p>聊天会话出现错误。</p>}
      <p>{connected ? '已连接到聊天。' : '已断开聊天连接。'}</p>

      <h3>可用操作</h3>
      <ul>
        {actions.map((action) => (
          <li key={action.id}>{action.name}</li>
        ))}
      </ul>

      {askUser && (
        <div>
          <h3>用户提示</h3>
          <p>{askUser.message}</p>
        </div>
      )}

      <h3>聊天设置</h3>
      <pre>{JSON.stringify(chatSettingsValue, null, 2)}</pre>
    </div>
  );
};
```

### `useChatInteract` Hook

`useChatInteract` hook 提供一组全面的方法，用于在 React 应用中与聊天系统交互。

#### 方法

- **`sendMessage`**
- **`replyMessage`**
- **`clear`**
- **`uploadFile`**
- **`callAction`**
- **`startAudioStream`**
- **`sendAudioChunk`**
- **`stopTask`**

#### 示例

```tsx
import { useChatInteract } from '@chainlit/react-client';

const ChatInteraction = () => {
  const { sendMessage, replyMessage, clear } = useChatInteract();

  return (
    <div>
      <button onClick={() => sendMessage({ content: '你好！' })}>发送</button>
      <button onClick={() => replyMessage({ content: '回复！' })}>回复</button>
      <button onClick={clear}>清除</button>
    </div>
  );
};
```

### `useAuth` Hook

`useAuth` hook 管理 React 应用中的[身份认证](/authentication)，提供用户[会话](/concepts/user-session)和令牌管理等功能。

#### 属性与方法

- **`authConfig`**
- **`user`**
- **`accessToken`**
- **`isLoading`**
- **`logout`**

#### 示例

```tsx
import { useAuth } from '@chainlit/react-client';

const UserProfile = () => {
  const { user, logout } = useAuth();

  if (!user) return <p>没有用户登录。</p>;

  return (
    <div>
      <p>用户名：{user.username}</p>
      <button onClick={logout}>退出登录</button>
    </div>
  );
};
```

### `useApi` Hook

`useApi` hook 使用 [SWR](https://swr.vercel.app/) 简化数据获取和错误处理。

#### 示例

```tsx
import { useApi } from '@chainlit/react-client';

const Settings = () => {
  const { data, error, isLoading } = useApi('/project/settings');

  if (isLoading) return <p>加载中...</p>;
  if (error) return <p>错误：{error.message}</p>;

  return <pre>{JSON.stringify(data, null, 2)}</pre>;
};
```
