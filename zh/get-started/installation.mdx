---
title: 安装
---

Chainlit 需要 `python>=3.9`。

你可以通过 pip 安装 Chainlit：

```bash
pip install chainlit
```

这会在你的系统上提供 `chainlit` 命令。

确认一切运行正常：

```bash
chainlit hello
```

这应会启动 Chainlit UI，并提示你输入姓名，如下所示：
![Hello](/images/hello.png)
