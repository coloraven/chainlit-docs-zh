Chainlit 是一个开源 Python 包，用于构建可用于生产环境的对话式 AI 应用。

<Frame caption="使用 Chainlit 构建对话式 AI">
  <video controls autoPlay loop muted src="/images/overview.mp4" />
</Frame>

## 核心特性

1. [快速构建：](/examples/openai-sql) 只需几行 Python 代码即可上手

2. [身份认证：](/authentication/overview) 与企业身份提供商及现有认证基础设施集成

3. [数据持久化：](/data-persistence/overview) 收集、监控并分析用户数据

4. [可视化多步推理：](/concepts/step) 一目了然地理解产生输出的中间步骤

5. [多平台：](/deploy/overview) 一次编写助手逻辑，随处使用
